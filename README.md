# REST API Post Manager

A simple React application built for Task 4 of the FlexiSAF Frontend Internship.

## Live Demo

https://api-user-dashboard-ashy.vercel.app/

## Project Overview

This project demonstrates how to consume and interact with a RESTful API in React using the four primary HTTP methods:

- GET
- POST
- PUT
- DELETE

The application uses the JSONPlaceholder API to fetch, create, update, and delete post data.

## Features

- Fetch posts from an API
- Create a new post
- Update an existing post
- Delete a post
- Loading state handling
- Error state handling
- Responsive design

## API Endpoint

```txt
https://jsonplaceholder.typicode.com/posts
```

## REST Methods Demonstrated

### GET
Retrieves posts from the API.

### POST
Creates a new post.

### PUT
Updates an existing post.

### DELETE
Removes a post.
